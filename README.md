Pet Pulse is a Pet Care and Adoption web application.

Frontend:
- HTML, CSS, JavaScript
- Hosted using GitHub Pages

Backend:
- Node.js
- Express.js
- MySQL
- Runs locally

Features:
- Pet listing and adoption enquiry
- Pet food products
- Image upload using Multer
- Cart and order management
- Admin product management

Note:
Backend APIs and database run locally.
Frontend is deployed live for demonstration.
